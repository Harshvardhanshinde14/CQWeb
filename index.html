FEATURES

CALCULATION OF QUANTITY

CalQuan KrossX generates in-depth quantity reports for each layer in the cross sections. Area definitions can be user-specific and also software-generated.

CROSS SECTION GENERATION

Detailed chainage-wise cross-sections are generated which are good for client submission. Also, with a single click, all drawings can be converted to PDF.

LINEAR PLANNING

Chainage & Time-based 2D planning is provided which is an innovative approach compared to existing project management software.

WORK PROGRESS TRACKING

In-built software modules for DPR, RFI entry which also generates graphical presentation for comparison between multiple phases such as Base Plan, DPR, RFI, etc.

RESOURCE TRACKING

In-built software modules for daily entry for consumption tracking of various resources such as machinery, manpower & materials.

SURVEY & DESIGN

Software modules for contour generation, design alignment, open plot volumes, Plan & Profile generation.

DYNAMIC 3D MODELLING

Dynamic 3D modelling based on input data. The 3D modelling is also integrated with work progress in order to generate Progressive 3D model.

MOBILE APP

Mobile App for DPR, RFI, Plant Production, Machine Log, and general Data Entry integrated with Desktop Application for Dynamic Updates and Reports.

MISCELLANEOUS

Other miscellaneous features for rate analysis, generation of strip charts, bar charts, Lab Tests, document management, mobile application, etc.

History

CalQuan is a leading software development company for civil engineering in India. CalQuan develops software for linear type projects like roads, railways, highways, airports, canals, dams and so on. CalQuan is a consolidated integrated software that includes in-depth solutions for quantity calculation, cross-section generation, planning, resource tracking, work-in-progress monitoring, document management, design, and survey. From Kashmir to Kanyakumari and Porbandar to Assam, CalQuan has a pan-India Installation base. CalQuan's software products are used by almost every major infrastructure company in India.
